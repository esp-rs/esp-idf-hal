\brief Configure SSL session ticket callbacks (server only). (Default: none.)

\note On server, session tickets are enabled by providing non-NULL callbacks.

\note On client, use \c mbedtls_ssl_conf_session_tickets().

\param conf SSL configuration context \param f_ticket_write Callback for writing a ticket \param f_ticket_parse Callback for parsing a ticket \param p_ticket Context shared by the two callbacks
