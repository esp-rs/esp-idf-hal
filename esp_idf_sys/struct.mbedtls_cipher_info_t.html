Expand description

Cipher information. Allows calling cipher functions in a generic way.

\note The library does not support custom cipher info structures, only built-in structures returned by the functions mbedtls_cipher_info_from_string(), mbedtls_cipher_info_from_type(), mbedtls_cipher_info_from_values(), mbedtls_cipher_info_from_psa().

\note Some fields store a value that has been right-shifted to save code-size, so should not be used directly. The accessor functions adjust for this and return the “natural” value.
